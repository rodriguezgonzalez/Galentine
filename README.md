# Galentine - Tu ne peux pas échapper à l'amitié

Un mini-jeu interactif pour la Galentine's Day, fait en HTML/CSS/JS pur.

## Le concept

Le jeu pose une question simple : **"Veux-tu être ma Galentine ?"**

- Si tu cliques **OUI** : victoire instantanée !
- Si tu essaies de cliquer **NON** : le bouton s'enfuit, orbite autour de OUI, puis te lance un défi...

### Phase 2 : Évite les bons sentiments

Un mini-jeu de type "dodge" où des sentiments tombent du ciel. Tu dois les éviter... mais le jeu est truqué : l'amitié monte quoi qu'il arrive. A la fin, tu deviens Galentine de toute façon.

**Contrôles :** Flèches / souris / tactile

## Stack

- HTML / CSS / JavaScript vanilla
- Police : [Nunito](https://fonts.google.com/specimen/Nunito) (Google Fonts)
- Aucune dépendance

## Lancer le jeu

Ouvre `index.html` dans un navigateur. C'est tout.
